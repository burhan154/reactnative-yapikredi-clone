## Installation				
				
### Running the project

Clone this repository :				
```
git clone https://github.com/burhan154/reactnative-yapikredi-clone.git
cd reactnative-yapikredi-clone
```				
				
Install packages :

```
npm install
```				
				
And run :

```bash
react-native run-ios
# or
react-native run-android
```
